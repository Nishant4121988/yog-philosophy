TikTok API Integration

Yog Philosophy uses TikTok’s official developer APIs to let creators authenticate and publish content directly to their TikTok profiles.

Our integration uses Login Kit for secure OAuth authorization and the Content Posting API to upload and publish videos on behalf of authorised users.

All API calls are made using TikTok’s sandbox environment for demonstration and testing purposes.

🔐 Connect with TikTok

Click the button below to authorise Yog Philosophy to access your TikTok account. This uses TikTok’s official OAuth 2.0 Login Kit flow.

Permissions Requested

  • user.info.basic — Read your basic profile info (name, avatar)
  • video.upload — Upload videos to your TikTok account
  • video.publish — Publish uploaded videos to your profile
You will be redirected to TikTok to authorise access. After approval, you will be redirected to this site.
